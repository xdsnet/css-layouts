利用css的grid显示特性，实现圣杯布局（页眉、中栏、页脚，其中中栏又分左边栏，中间主要内容，右边栏）。

     内容包装部分样式（实现页面、中、页脚）
.parent {
    display: grid;
    grid-template: auto 1fr auto / auto 1fr auto;
}
  
header {
    grid-column: 1 / 4;
}
.left-side {
    grid-column: 1 / 2;
}
main {
    grid-column: 2 / 3;
}

 .right-side {
    grid-column: 3 / 4;
}

footer {
    grid-column: 1 / 4;
}
