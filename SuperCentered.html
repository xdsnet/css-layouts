利用css的grid显示特性，一句就可以实现居中（水平和垂直都居中）、起始（左上角）和末尾（右下角）对齐。

            .mycenter {
                display: grid;
                place-items: center;
            } 
            .mystart {
                display: grid;
                place-items: start;
            } 
            .myend {
                display: grid;
                place-items: end;
            }
