利用css的grid和flex显示特性，实现多并列卡片布局。

    外部包装样式
    .parent {
        height: auto;
        display: grid;
        grid-gap: 1rem;
        grid-template-columns: repeat(3, 1fr);
    }
    
    .visual {
        height: 100px;
        width: 100%;
    }
    
    .card {
        display: flex;
        flex-direction: column;
        padding: 1rem;
        justify-content: space-between;
    }
    
    h3 {
        margin: 0
    }
